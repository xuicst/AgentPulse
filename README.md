# Installing AgentPulse from a VSIX Package
## Method 1: Install via the VS Code UI (Recommended)

1. Open **Visual Studio Code**.
2. Go to the **Extensions** view (`Ctrl + Shift + X`).
3. Click the **...** (More Actions) button in the top-right corner.
4. Select **Install from VSIX...**.
5. Choose the **`AgentPulse-1.0.0.vsix`** file.
6. Once the installation is complete, reload VS Code if prompted.


## Method 2: Install via the Command Line

Open a terminal and run:

```bash
code --install-extension AgentPulse-1.0.0.vsix
```

If the file is located elsewhere, specify its full path:

```bash
code --install-extension "D:\AgentPulse\AgentPulse-1.0.0.vsix"
```

Restart VS Code after the installation if necessary.


## Verify the Installation

After installation:

1. Open the **Extensions** view.
2. Search for **AgentPulse**.
3. Ensure the extension is installed and enabled.

You are now ready to use AgentPulse.


## Updating the Extension

To update AgentPulse, simply install the newer `.vsix` package using the same steps above. VS Code will automatically replace the existing version.
